# Day 32 Log

**Date:** May 21, 2026

**Focus:** Multi-Modal LLMs (Vision + Language)

**What I Did**
- Learned about multi-modal models that combine text and images
- Tested image captioning and visual question answering
- Used BLIP models for image understanding
- Explored how vision and language are integrated in LLMs

**Key Learnings**
- Multi-modal models can understand both text and images
- BLIP and LLaVA are popular for image-to-text tasks
- Visual Question Answering (VQA) is a powerful capability
- These models are useful for real-world applications like accessibility and content description

**Challenges**
- Loading images from URLs (network issues)
- Model size and speed on local machines
- Limited context understanding in small multi-modal models

**Reflection**
Multi-modal LLMs are the future of AI. Being able to combine vision and language opens many new applications (image description, visual search, accessibility tools, etc.).
