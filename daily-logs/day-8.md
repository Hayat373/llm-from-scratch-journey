# Day 8 Log

**Date:** June 10, 2026

**Focus:** Transition from Building from Scratch to Using Pre-trained Models (Hugging Face)

**What I Did**
- Completed 7 days of building MiniGPT from scratch (Bigram → Self-Attention → Full Transformer)
- Started using Hugging Face Transformers library
- Loaded and ran real pre-trained models like GPT-2
- Realized the huge difference between from-scratch models and real LLMs

**Key Learnings This Week**
- Deeply understood Transformer architecture (Attention, Multi-Head, Causal Masking, Residuals, LayerNorm)
- Learned how to build, train, save, and generate from a small GPT-like model
- Discovered that using pre-trained models (Hugging Face) is much more practical and powerful
- From-scratch learning gave me strong fundamentals, but real-world work uses pre-trained models


**Honest Challenges**
- I was mostly copying and modifying code rather than writing it independently
- My from-scratch models produce very limited text quality compared to even GPT-2
- Training small models from scratch is educational but not efficient 
- I still need a lot of practice to code Transformer components from memory


**Big Realization**
I don't need to build LLMs from scratch for jobs or hackathons. Strong understanding + ability to use, fine-tune, and build applications on top of pre-trained models is what matters most.
