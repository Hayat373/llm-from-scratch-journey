# Day 10 Notes - Advanced Prompt Engineering

**Date:** Jun 11, 2026

## Key Concepts Learned

### 1. Prompt Engineering Techniques
- **System Prompt**: Defines the role and behavior of the AI (most important)
- **Chain of Thought (CoT)**: Asking the model to "think step by step"
- **Few-shot Prompting**: Giving examples in the prompt
- **Role Playing**: Making the model act as a specific character/expert
- Generation parameters (`temperature`, `top_p`, `top_k`)

### 2. Best Practices
- Clear and specific system prompts work better
- Context matters — Ethiopian-focused prompts gave more relevant answers
- Separating system and user messages improves structure
